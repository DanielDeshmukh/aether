# AETHER Engineering Protocols

## Architectural Standards
- **Pattern:** Multi-agent autonomous loops with deterministic state validation.
- **Security:** Every module must implement the "AETHER-Shield" middleware for token validation.
- **Routing:** Use the "Intent-Router" logic developed for Hector (deterministic schema-based routing).


## Definition of Done
- Jules must generate a unit test in `/tests` for every new function.
- All code must pass a mobile-responsive audit for the dashboard.
